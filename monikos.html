overview

Monikos is a webview mobile app for Android and iOS built for pharmacy students to help memorize the overwhelming drug facts they have to remember in pharmacy school. This was an Agile team project for a teacher supervised Software Engineering Lab where I worked together with three other students and a client.

What I did:

  • Collaborated with client to determine app functionalities and structure
  • Created a "drugs database" section with drugs information retrieved from a MySQL database using AngularJS, PHP, and SQL
  • Developed like/dislike and suggestion functionality for mnemonic hints using angular's $http service and PHP
  • Created popup menu to hold local user information throughout the app using cookies
  • Created a list manager to create personalized custom drugs list
  • Pulled stored "university lists" into user's local collection using PHP and SQL
  • Communicated with teammates and client through Slack and weekly meetings

in-depth look

To simplify operations, we divided the app into 2 main functions:
1. View quick information on specific drugs through a drugs "database" section
2. "Study" from your school and personal lists of drugs with games

We retrieved drug information from our MySQL database through an HTTP GET request and loaded the list of JSON encapsuled drug information using ngRepeat. Using AngularJS' filter module we could search through the drugs.

Monikos established itself apart from other pharmaceutical study aid apps by its database of mnemonic hints for each drug. HTTP POST requests allow for liking and disliking hints. You can also send in suggestions through a PHP form that is sent through e-mail for manual filtering.

When a user registers they were asked to affiliate themselves with a university. Each university has a collection of ready-to-use lists. You can also make your own custom drugs list from the database. Checklist-Model allowed us to easily create list objects. Using the lists you can play games to earn "capsules" and wager your "capsules" with your friends.
