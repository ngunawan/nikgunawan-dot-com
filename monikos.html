overview

Monikos is a webview mobile app for Android and iOS built for pharmacy students to help memorize the overwhelming drug facts they have to remember in pharmacy school. This was an Agile team project for a Software Engineering Lab course where I worked together with three other students and two clients.

What I did:

  • Collaborated closely with clients in negotiating, estimating, and scheduling end-product.
  • Worked in an Agile environment with weekly meetings and bi-weekly sprint goals.
  • Created a drugs database section by utilizing various data from a MySQL database using PHP and SQL.
  • Created a list manager to create and manage custom drugs list objects.
  • Developed like/dislike and suggestion functionality.
  • Created a user menu to hold local user information using cookies.

in-depth look

To simplify operations, we divided the app into two main sections:
1. Database section - to quickly lookup drugs information.
2. Study section - where you learn from pre-made and custom-made drugs list by playing mini games.

We retrieved drug information from our MySQL database through an HTTP GET request and loaded the list of JSON encapsulated drug information using ngRepeat. Using AngularJS' filter we could search through the drugs.

Monikos established itself apart from other pharmaceutical study aid apps with a collection of mnemonic hints available for each drug. HTTP POST requests allow for liking and disliking hints. You can also send in hint suggestions through a PHP form.

When a user registers, they are asked to affiliate themselves with a university. Each university has a collection of ready-to-use lists. You can also make your own personal custom drugs list from the database using the list manager. Using the checklist-model directive by vitalets allowed us to easily create list objects in AngularJS. Using the lists you can play games to earn "capsules" and wager your "capsules" with your friends.
