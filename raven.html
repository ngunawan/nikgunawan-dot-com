overview

RavenNicole.com was created for an industrial dancer designed to showcase her past works and get hired by talent agencies. Her old site was created years ago and she wanted a more modern and stylish look. I worked on this project with a fellow UX designer to renovate her old site.

What I did:

  • Built site from scratch using HTML, CSS and JQuery.
  • Translated Photoshop mockups designed by UX designer to a pixel perfect site.
  • Used JQuery's mouse events to create an interactive choreography gallery.
  • Used magnific-popup.js to implement youtube videos without leaving the site.
  • Used PHP to create a contact form.
  • Designed site to be responsive on any size device up to 1920x1080.
